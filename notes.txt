cd ../..                        - перейти на 2 папки выше
ls                              - содержимое папки
dbt-env\Scripts\activate        - активировать виртуальное окружение
dbt init dbt_project            - создать проект
dbt debug                       - проверить соединение с бд и что с проектом все окружение
dbt build                       - собрать проект

Дальше команды гит

git init                                                                - чтобы создать гит репозиторий
git branch -M main                                                      - чтобы сменить ветку
git remote add origin https://github.com/alrsho/dbt_project.git         - привязать к онлайн репо
git push -u origin main                                                 - пушим коммит (создали его с помощью vsc)


Дальше нам необходимо создать источники в dbt - их будем создавать в папке models=>staging (для каждого источника своя папка)
источники прописываются при помощи файлов yml (задаются их параметры)

После прописывания источников можно создавать модели с данными из них



Для проерки свежести данных (по настройкам из yml) команда:
dbt source freshness
